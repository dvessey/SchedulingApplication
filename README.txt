DIRECTIONS TO RUN PROGRAM
1) Login with username "test" and password "test"
2) Click Add customer
	-fill out all information
3) Click cancel to return to appointment screen
4) Add appointment
	-fill out required information
5) On the customer screen, you can click a customer then click show selected customer's appointments
	to show all appointments associated with that customer
6) Click reports button to be taken to the report screen to select the report you want to view
======================================================================================================
